sensor data
===========

These are measurements of a **translation along the z axis** with the **z axis pointing up**.

    Z = moved, up
    |
    |  Y
    | /
    |/______X

### InvenSense MPU6050 ###

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

### Honeywell HMC5883L ###

- Magnetometer

#### Speed ####

- measured at 75Hz
