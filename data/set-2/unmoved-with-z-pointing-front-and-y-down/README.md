sensor data
===========

These are measurements of an **unmoved system** with the **z axis pointing front**.

    
                    Z = fix, front
                     \
                      \
     X fix, left ______\
                       |
                       |
                       |
                       Y = fix, down

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

#### Speed ####
