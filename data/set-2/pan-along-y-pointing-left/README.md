sensor data
===========

These are measurements of a **translation along the y axis** with the **y axis pointing left**.

    Z
    |  Y = moved, left
    | /
    |/______X

### InvenSense MPU6050 ###

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

### Honeywell HMC5883L ###

- Magnetometer

#### Speed ####
