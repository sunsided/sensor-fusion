sensor data
===========

These are measurements of a **rotation around the z axis** with the **z axis pointing up**.

    Z = fix, up
    |
    |  Y (left)
    | /
    |/______X (forward)

### InvenSense MPU6050 ###

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

### Honeywell HMC5883L ###

- Magnetometer

#### Speed ####

- measured at 75Hz
