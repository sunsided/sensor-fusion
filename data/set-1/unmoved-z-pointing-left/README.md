sensor data
===========

These are measurements of an **unmoved system** with the **z axis pointing left**.

    
    
       Z = fix, left
      /
     /______X = fix, forward
    |
    |
    |
    Y = fix, down

### InvenSense MPU6050 ###

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

### Honeywell HMC5883L ###

- Magnetometer

#### Speed ####

- measured at 75Hz
