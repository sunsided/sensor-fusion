sensor data
===========

These are measurements of a **rotation around the x axis** with the **x axis pointing forward**.

    Z
    |  Y
    | /
    |/______X = fix, forward

### InvenSense MPU6050 ###

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

### Honeywell HMC5883L ###

- Magnetometer

#### Speed ####

- measured at 75Hz
