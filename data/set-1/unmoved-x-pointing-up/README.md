sensor data
===========

These are measurements of an **unmoved system** with the **x axis pointing up**.

                   X = fix, up
                   |
                   |
                   |
    Y = fix  ______|
        left      /
                 /
                /
               Z = fix, front

- Accelerometer
- Gyroscope
- Temperature

#### Speed ####

- measured at 100Hz

### Honeywell HMC5883L ###

#### Speed ####
